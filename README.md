# PySecurity
The fastest and most practical tool for DNS and IP check operations. Created using Hackertarget API.

## Installation
```py
git clone https://github.com/pycompiler/PySecurity.git && cd PySecurity
```
```py
pip install -r requirements.txt
```
```py
py pysecurity.py 
```
